## 1 简介

项目一个阶段开发完成，要发布了，可以使用 git tag 打标签。

## 2 常用操作

```java
//1.打标签
git tag -a v1.0 //-a 创建一个带注解的标签

//2.追加标签
git tag -a v1.1 64ee //如果忘了给某个提交打标签，又将它发布了，我们可以给它追加标签

//3.查看所有标签
git tag

//4.删除标签
git tag -d v1.0

//5.查看此版本所修改的内容
git show v1.1    

//6.指定标签信息
git tag -a [name] -m "信息"
//例
git tag -a v2.0 -m "支付功能完成发布"    
```
